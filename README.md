# Comparative Recommendation System

A React-based comparative recommendation system that allows users to analyze and compare multiple recommendation algorithms.

## Features

- **4 Recommendation Algorithms:**
  - Content-Based Filtering
  - Collaborative Filtering
  - Hybrid Approach
  - Multi-Armed Bandit (MAB)

- **Dynamic Metrics:**
  - Precision, Recall, F1-Score
  - Satisfaction, Diversity, Coverage
  - NDCG (Normalized Discounted Cumulative Gain)

- **Interactive UI:**
  - Algorithm comparison visualizations
  - Dataset viewer
  - Real-time metric computation

## Installation
```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/recommendation-system.git

# Navigate to project directory
cd recommendation-system

# Install dependencies
npm install

# Start the development server
npm start
```

## Technologies Used

- React
- Recharts (Data Visualization)
- Lucide React (Icons)
- Tailwind CSS (Styling)

## Usage

1. Select a user from the dropdown
2. Choose an algorithm or compare all
3. View recommendations and performance metrics
4. Analyze algorithm differences
